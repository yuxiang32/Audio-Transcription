# Audio-Transcription

This is a side project I built showcasing OpenAI Whisper transcription capabilities, I built it using Python, Anaconda and Streamlit. 

https://user-images.githubusercontent.com/3026442/213085165-e1455e25-1212-4095-93a0-97c0417c002e.mp4

In order to run this program, you will need to install a few dependencies

On Linux
`sudo apt update && sudo apt install ffmpeg`

On MacOS
`brew install ffmpeg`

On Windows
`chco install ffmpeg`

Install pytorch
`conda install pytorch torchvision torchaudio pytorch-cuda=11.7 -c pytorch -c nvidia`

Install Whisper
`pip install git+https://github.com/openai/whisper.git -q`

Install streamlit
`pip install streamlit`
